# Computer Architecture · Interactive

**From logic gates to pipelined processors**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 3 · Undergraduate

## What You'll Learn

- Boolean algebra and logic gates
- Combinational and sequential circuits
- Assembly language and ISA (RISC-V)
- Processor datapath and control
- Pipelining and hazards
- Memory hierarchy: caches, virtual memory

## References

- David A. Patterson & John L. Hennessy, *Computer Organization and Design* (6th ed., Morgan Kaufmann)
- Noam Nisan & Shimon Schocken, *The Elements of Computing Systems* (Nand2Tetris)
- Charles Petzold, *Code: The Hidden Language of Computer Hardware and Software* (2nd ed.)

## Features (when built)

- **Interactive visualizations** — every concept comes alive with draggable, animated demos
- **KaTeX math rendering** — rigorous notation throughout
- **Exercises with hints & solutions** — from computational to proof-based
- **Progress tracking** — saved locally in your browser
- **Zero dependencies** — just open the link and learn
